📖 使用说明 - SSE 版本

技术方案

  • 连接方式: Server-Sent Events (SSE) - 单向服务器推送
  • 音频播放: 客户端直接播放原始音频流
  • 字幕同步: 根据 Azure 返回的时间戳 (offset) 同步显示字幕

工作流程

  1. 调用 /sse/connect 获取连接 token
  2. 使用 token 连接 SSE 端点
  3. 连接成功后调用 /sse/subscribe 启动电台
  4. 通过 SSE 实时接收字幕数据
  5. 根据音频播放时间匹配显示字幕

优势

  • ✅ 标准 HTTP 协议，无需 WebSocket
  • ✅ 自动重连机制
  • ✅ 更好的防火墙兼容性
  • ✅ 低延迟字幕推送
